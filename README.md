# Traducciones-Fivem #
Todas los locale.lua en español

Una vez añadido todo a sus respectivos lugares. Añadir en __resources.lua
```lua
server_scripts {
	'locales/es.lua',
}

client_scripts {
	'locales/es.lua',
}
```
Despues de eso, modificar "Config.Locale = 'en'" en el archivo config.lua de cada plugin: 
```lua 
Config.Locale = 'es'
```
